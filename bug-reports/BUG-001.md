# BUG-001

Title: Login fails without showing error message

Environment:
- Browser: Chrome 125
- OS: Windows 11
- URL: https://www.saucedemo.com/

Severity: High
Priority: High
Reproducibility: 100%

Preconditions:
- User is on login page

Steps to Reproduce:
1. Open login page
2. Enter valid username: standard_user
3. Enter invalid password: wrong_password
4. Click Login button

Actual Result:
- Page reloads OR stays on same page
- No error message is displayed to user

Expected Result:
- System should display an error message like:
  "Epic sadface: Username and password do not match any user in this service"
